Environmental Assessment of Zoobenthic Communities

A computational framework for sediment contamination assessment and community-condition indexing

This project develops a reproducible analytical pipeline for assessing the ecological condition of zoobenthic (bottom-dwelling invertebrate) communities in relation to sediment contamination. The work integrates three types of site-level data—environmental habitat descriptors, sediment chemistry, and taxonomic community composition—to build a bioassessment framework that separates natural environmental variation from contamination-driven impacts.

The pipeline processes 310 sites with complete data across three matrices:
